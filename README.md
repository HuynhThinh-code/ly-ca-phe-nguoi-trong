# Fair Bean Chain

Trang web thuyet trinh tuong tac ve quan he loi ich kinh te trong chuoi gia tri ca phe Viet Nam.

## Chay cuc bo

Mo truc tiep `index.html` trong trinh duyet, hoac chay:

```bash
npm run build
```

## Deploy GitHub Pages

1. Tao repository moi tren GitHub.
2. Push cac file trong thu muc nay len nhanh `main`.
3. Vao `Settings -> Pages`.
4. Chon `Deploy from a branch`, branch `main`, folder `/root`.

Trang chinh la `index.html`.
